PHP

PHP é uma linguagem de programação voltada para o desenvolvimento web, executada no servidor.

Uma das suas principais funções é a comunicação com bancos de dados, como o PostgreSQL.

Isso permite armazenar, buscar e manipular informações dinamicamente.

Para gerenciar o banco de dados visualmente, usamos o pgAdmin, que permite criar tabelas, visualizar registros e executar comandos SQL de forma gráfica.

Já a conexão entre o PHP e o PostgreSQL é feita através da variável $conn, que guarda as informações do banco.

Sempre envolva a conexão em um bloco try/catch para capturar erros sem expor dados sensíveis.

Para executar consultas no banco, usamos a variável $varSQL para guardar o comando SQL escrito.

Em seguida, o método prepare prepara esse comando antes de enviá-lo ao banco; isso evita erros e aumenta a segurança.

Depois, o bindParam é usado para vincular os valores das variáveis PHP aos parâmetros do SQL, substituindo os ? ou :nome pelos dados reais.

Por fim, o execute envia o comando ao banco, e o fetch recupera os resultados linha por linha.

Com a conexão estabelecida pelo $conn, podemos aplicar o CRUD, as quatro operações fundamentais de qualquer sistema:

Create (inserir),
Read (buscar),
Update (atualizar) e
Delete (deletar) registros no banco.

Essas operações são organizadas em funções, deixando o código mais limpo, reutilizável e fácil de manter.

Dentro dessas funções, o fluxo padrão é:

1. Definir o SQL na $varSQL;

2. Usar o prepare para preparar o comando;

3. Utilizar o bindParam para passar os valores com segurança;

4. Executar com execute.

Quando precisamos recuperar dados, o fetch ou fetchAll retorna os registros do banco para o PHP.

Esse fluxo protege o sistema contra ataques de SQL Injection e mantém o código organizado.
