HTML, CSS e JavaScript

HTML, CSS e JavaScript são as três linguagens base do desenvolvimento web.
O HTML estrutura o conteúdo da página, o CSS cuida da aparência visual e o JavaScript adiciona interatividade e comportamento.

No CSS, os layouts definem como os elementos são organizados na tela.
Os dois sistemas de layout mais usados são o Flexbox, ideal para alinhar elementos em linha ou coluna, e o Grid, que organiza o conteúdo em linhas e colunas ao mesmo tempo.
Com eles conseguimos criar páginas responsivas que se adaptam a diferentes tamanhos de tela.

O CSS também permite criar animações básicas com duas abordagens principais.
A propriedade transition suaviza mudanças de estilo ao interagir com um elemento, como ao passar o mouse.
Já o @keyframes permite criar animações mais completas, definindo etapas do movimento com from e to, ou usando porcentagens como 0%, 50% e 100%.

O JavaScript é executado diretamente pelo navegador e permite manipular a página em tempo real.
O primeiro contato com JavaScript costuma ser pelo console do navegador, acessado com F12.
Por lá é possível testar comandos, ver erros e usar o console.log() para exibir valores e mensagens durante o desenvolvimento.

Juntos, HTML, CSS e JavaScript formam a base de qualquer página web moderna.
O HTML organiza, o CSS estiliza e anima, e o JavaScript torna tudo dinâmico e interativo.
O console é uma ferramenta essencial nesse processo ele permite testar código rapidamente, identificar erros e entender o comportamento do programa antes de aplicá-lo na página.
