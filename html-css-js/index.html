HTML, CSS e JavaScript

São as três linguagens base do desenvolvimento web. O HTML estrutura o conteúdo, o CSS define a aparência e o JavaScript adiciona interatividade (veja as camadas abaixo).

Layout no CSS

Os layouts definem como os elementos se organizam na tela. Os dois sistemas mais usados:

  • Flexbox — alinha elementos em linha ou coluna, de forma simples.
  • Grid — organiza em linhas e colunas ao mesmo tempo.

Com eles criamos páginas responsivas, que se adaptam a diferentes tamanhos de tela.

Animações no CSS

  • transition — suaviza mudanças de estilo ao interagir (ex.: a cor de um botão no hover).
  • @keyframes — animações mais completas, com etapas definidas por from/to ou 0%, 50%, 100%.

JavaScript & console

  • Executado pelo navegador, manipula os elementos da página em tempo real.
  • O console (tecla F12) é o primeiro contato: testar comandos e ver erros.
  • console.log() exibe mensagens e valores durante o desenvolvimento.

O console é essencial para testes e depuração — ajuda a entender o código antes de aplicá-lo na página.
