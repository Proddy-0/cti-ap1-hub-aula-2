Java

Java é uma linguagem de programação orientada a objetos, amplamente usada no desenvolvimento de sistemas, aplicativos e back-end web.
É uma linguagem fortemente tipada, ou seja, todo dado precisa ter seu tipo definido como int, String, double antes de ser usado.

Um dos recursos mais importantes do Java é o try-catch, usado para tratamento de erros.
O bloco try contém o código que pode gerar um erro, e o bloco catch captura esse erro e define o que fazer com ele.
Isso evita que o programa trave inesperadamente e permite exibir mensagens de erro mais claras para o usuário.

Para trabalhar com números aleatórios, o Java oferece a biblioteca java.util.Random.
Com ela, criamos um objeto da classe Random e usamos métodos como nextInt(), nextDouble() e nextBoolean() para gerar valores aleatórios.
É muito usada em jogos, sorteios e simulações.

Os arrays são estruturas que armazenam múltiplos valores do mesmo tipo em uma única variável.
Em Java, o tamanho do array é definido na criação e não pode ser alterado depois.
Acessamos cada elemento pelo seu índice, que começa sempre em 0.

Combinando arrays, Random e try-catch, conseguimos criar programas mais robustos e dinâmicos.
Por exemplo, podemos preencher um array com números aleatórios e usar o try-catch para evitar erros ao acessar posições inválidas como um índice fora do tamanho do array, o que geraria um ArrayIndexOutOfBoundsException.
Tratar esses erros corretamente é uma das boas práticas fundamentais em Java.
