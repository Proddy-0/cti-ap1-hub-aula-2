Java

Java é uma linguagem orientada a objetos, usada em sistemas, aplicativos e back-end web. É fortemente tipada: todo dado precisa ter seu tipo definido (int, String, double) antes de ser usado.

Tratamento de erros — try-catch

  • try — contém o código que pode gerar uma exceção.
  • catch — captura o erro e define o que fazer quando ele ocorre.
  • Evita que o programa seja encerrado inesperadamente e permite mensagens mais claras ao usuário.

Números aleatórios — Random

A biblioteca java.util.Random gera valores aleatórios a partir de um objeto da classe Random. Métodos mais usados:

  • nextInt() — números inteiros.
  • nextDouble() — números decimais.
  • nextBoolean() — verdadeiro ou falso.

Muito utilizada em jogos, sorteios e simulações.

Arrays

  • Armazenam vários valores do mesmo tipo em uma única variável.
  • O tamanho é definido na criação e não pode ser alterado depois.
  • Cada elemento é acessado por um índice — que sempre começa em 0.
